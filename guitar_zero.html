Guitar Zero

Source code: still need to upload it.

This was a 5-person group project in my second year for the University of Exter. We were given a plastic guiter controller, and instructed to make a clone of guitar Hero. We built a 3D re-creation of guitar hero, including a tuturial and a server from which you could download music.

Given that this was a second-year university project it isn't exactly relevant to anything anymore, but I still found it a fun / cool project and wanted to highlight it. Especially since it had to be completed in a matter of a few weeks.
